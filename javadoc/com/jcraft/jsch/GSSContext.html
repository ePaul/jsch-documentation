Interface GSSContext

public interface GSSContext

Usually not to be used by applications.

Encapsulates a GSS context, i.e. an implementation of the GSS API. This is used by UserAuthGSSAPIWithMIC to do the actual cryptographic work. (See this class for a description on how this interface is used.)

This interface only contains the methods needed for GSS-API authentication, not the full GSS API.

An implementation of this interface is included in the JSch library, based on the GSS API included in the Java SE from 1.4 on, using object ID 1.2.840.113554.1.2.2, i.e. Kerberos v5 as defined in RFC 1964.

See Also:
GSSContext, RFC 2743, Generic Security Service Application Program Interface Version 2, Update 1, RFC 5653: Generic Security Service API Version 2: Java Bindings Update, RFC 4462: Generic Security Service Application Program Interface (GSS-API) Authentication and Key Exchange for the Secure Shell (SSH) Protocol, RFC 1964, The Kerberos Version 5 GSS-API Mechanism
