com.jcraft.jsch
Interface Logger

public interface Logger

Provides logging abilities to the library. An application might implement this interface to capture the logging data, or to send it to some destination.

The library provides a default implementation which sends the output to /dev/null, i.e. does not do any logging.

See Also:
JSch.setLogger(com.jcraft.jsch.Logger)
